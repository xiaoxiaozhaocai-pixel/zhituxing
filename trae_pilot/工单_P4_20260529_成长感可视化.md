## 背景

架构 2.0 决策：小职不是普通聊天机器人，是用户的 AI 朋友。用户需要感受到「陪伴在成长」——不是冷冰冰的工具，而是一起走过的时间。

本次只做**纯展示组件**，不改任何 API/DB/业务逻辑。组件放在个人中心新增「成长记录」tab 下，读取现有数据展示。数据缺失时优雅降级（空状态提示），绝不能白屏或报错。

### 样式要求

- 卡片用 `<Card>` 组件（来自 `@/components/ui/card`）
- 4 个数字统计用 grid 布局，2×2（移动端）或 1×4（桌面端）
- 数字用 `text-3xl font-bold`，标签用 `text-sm text-gray-500`
- 主色调 `#165DFF`（蓝），底部寄语区淡蓝背景 `bg-blue-50`
- 加载态：数字位置显示 `<Loader2 className="animate-spin" />`
- `companionDays = 0` 时：显示"今天是我们相识的第一天 🎊"

### 空状态

所有计数为 0 时显示"今天是我们相识的第一天 🎊 未来的每一步，小职都会陪你一起走"，带按钮 `<Link href="/">` 跳转首页。

## 任务 2：成长里程碑时间线组件

**文件**：`src/components/GrowthTimeline.tsx`（新建）

### Props

```ts
interface Milestone {
  id: string;
  type: 'register' | 'assessment' | 'report' | 'resume' | 'favorite' | 'interview';
  title: string;
  description: string;
  date: string;  // ISO date string
}

interface GrowthTimelineProps {
  milestones: Milestone[];
  isLoading?: boolean;
}
```

### type 对应图标和颜色

| type | 图标 | 颜色 | 标题模板 |
|------|------|------|---------|
| register | `Sparkles` | `#165DFF` | 加入职途星 |
| assessment | `Brain` | `#722ED1` | 完成胜任力测评 |
| report | `FileText` | `#10B981` | 生成{type}报告 |
| resume | `FileSearch` | `#F59E0B` | 优化简历 |
| favorite | `Heart` | `#EF4444` | 收藏了心仪岗位 |
| interview | `Mic` | `#8B5CF6` | 完成模拟面试 |

### 样式要求

- 左侧竖线用 `border-l-2 border-gray-200`，图标圆点 `w-3 h-3 rounded-full` + 对应颜色
- 每个里程碑：日期用 `text-xs text-gray-400`，标题用 `font-medium`，描述用 `text-sm text-gray-500`
- **只显示最近 10 条**，超出不显示
- 加载态：3 个 `<div className="animate-pulse bg-gray-200 h-12 rounded" />` 骨架屏
- 空状态：只有 1 条注册里程碑 + "更多精彩，等你探索 ✨"

    setGrowthLoading(true);

  }
}, [activeTab]);
