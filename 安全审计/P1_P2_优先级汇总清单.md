### 第一天（上线前必修，约 1.5 小时）
```

```

### 第二天（建议上线前，约 2 小时）
```
4. [10min] 修复 P1-MED-2：admin_token 加哈希存储

6. [10min] 修复 P2-MED-1：AIResponseRenderer 加 sanitizeHtml 保护
7. [20min] 修复 P2-MED-2：文章评论输入输出 XSS 防护
```
